# /psql-schema

Показать структуру объекта или схемы в PostgreSQL.

**Goal:** Быстро увидеть колонки/типы/индексы/FK/размер таблиц без ручного ввода `\d+`.

**Output:** Описание таблицы (колонки, индексы, FK, constraints, размер) или список таблиц схемы с размерами.

**Scenarios:**

- Без аргументов -- список таблиц текущей схемы (`\dt+`).
- `table` -- структура таблицы (`\d+ table`): колонки, типы, default, индексы, FK, partition info.
- `schema` (имя схемы) -- список таблиц схемы с размерами.
- `--indexes table` -- только индексы и их размер для таблицы.
- `--db name` -- подключиться к конкретной БД.

**Constraints:**

- Требует `psql` в PATH; если не найден -- показать инструкцию установки и ссылку на `docs/CLI_UTILITIES.md`.
- Read-only по природе (`\d`, `pg_catalog`); никаких изменений схемы не выполняется.
- Размеры таблиц считаются через `pg_total_relation_size` -- может быть медленно на больших инстансах, упомянуть в выводе.
