# dex-telegram-notifier

Плагин для отправки уведомлений в Telegram при работе Claude Code.

## Возможности

- **Stop** - Claude завершил работу
- **Notification** - Claude ждёт ответа пользователя
- **SubagentStop** - Субагент завершил подзадачу

### Содержимое уведомлений

- Последнее сообщение Claude
- Ultrathink (опционально)
- Статус TODO списка
- Используемые инструменты
- Текущий план (если есть)
- Вопросы к пользователю (AskUserQuestion)

## Установка

### 1. Создайте Telegram бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Сохраните полученный токен (формат: `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Получите Chat ID

1. Отправьте любое сообщение вашему новому боту
2. Откройте в браузере:
   ```
   https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
   ```
3. Найдите `"chat":{"id":123456789}` - это ваш Chat ID

### 3. Настройте переменные окружения

```bash
# Добавьте в ~/.bashrc, ~/.zshrc или .env файл проекта
export TELEGRAM_BOT_TOKEN="your-bot-token"
export TELEGRAM_CHAT_ID="your-chat-id"
```

### 4. Проверьте настройку

```bash
/notify-test
```

## Конфигурация

### Обязательные переменные

| Переменная | Описание |
|------------|----------|
| `TELEGRAM_BOT_TOKEN` | Токен бота от @BotFather |
| `TELEGRAM_CHAT_ID` | ID чата для уведомлений |

### Опциональные переменные

#### События (все включены по умолчанию)

| Переменная | Default | Описание |
|------------|---------|----------|
| `TELEGRAM_NOTIFY_STOP` | `true` | Уведомлять о завершении работы |
| `TELEGRAM_NOTIFY_WAITING` | `true` | Уведомлять об ожидании ответа |
| `TELEGRAM_NOTIFY_SUBAGENT` | `true` | Уведомлять о субагентах |

#### Компоненты сообщений

| Переменная | Default | Описание |
|------------|---------|----------|
| `TELEGRAM_INCLUDE_MESSAGE` | `true` | Включать последнее сообщение |
| `TELEGRAM_INCLUDE_THINKING` | `false` | Включать Ultrathink |
| `TELEGRAM_INCLUDE_TODO` | `true` | Включать статус TODO |
| `TELEGRAM_INCLUDE_TOOLS` | `true` | Включать используемые инструменты |
| `TELEGRAM_INCLUDE_PLAN` | `true` | Включать план |
| `TELEGRAM_INCLUDE_QUESTIONS` | `true` | Включать вопросы |

#### Прочие настройки

| Переменная | Default | Описание |
|------------|---------|----------|
| `TELEGRAM_LANGUAGE` | `ru` | Язык сообщений (`ru` или `en`) |
| `TELEGRAM_MESSAGE_LIMIT` | `4000` | Лимит символов в сообщении |
| `TELEGRAM_THREAD_ID` | - | ID топика для супергрупп |

## Команды

| Команда | Описание |
|---------|----------|
| `/notify-test` | Отправить тестовое уведомление |
| `/notify-config` | Показать текущую конфигурацию |

## Зависимости

- `curl` - для отправки HTTP запросов
- `jq` - для парсинга JSON

Обычно предустановлены в большинстве Linux/macOS систем.

## Устранение неполадок

### Уведомления не приходят

1. Проверьте переменные: `/notify-config`
2. Отправьте тест: `/notify-test`
3. Проверьте что бот не заблокирован в Telegram

### Ошибка "Unauthorized"

Неверный токен бота. Проверьте `TELEGRAM_BOT_TOKEN`.

### Ошибка "Chat not found"

Неверный Chat ID. Убедитесь что:
1. Вы отправили боту хотя бы одно сообщение
2. Chat ID скопирован правильно

### Ошибка "Bot was blocked by the user"

Разблокируйте бота в Telegram (найдите чат с ботом → Unblock).

## Безопасность

- **Никогда** не храните токен бота в репозитории
- Используйте переменные окружения или `.env` файлы
- Добавьте `.env` в `.gitignore`
