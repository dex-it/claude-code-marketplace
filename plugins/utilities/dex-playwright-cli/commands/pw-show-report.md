# /pw-show-report

Открыть HTML-отчёт последнего прогона тестов через `npx playwright show-report`.

**Goal:** Показать локальный URL отчёта с разбивкой по тестам, traces, attachments, видео и скриншотами.

**Output:** Локальный URL (`http://localhost:9323` по умолчанию), краткая выжимка из отчёта если доступна (число passed/failed по reporter-json). Подсказка `Ctrl-C` для остановки локального сервера.

**Scenarios:**

- Без аргументов -- отчёт по умолчанию (`playwright-report/`).
- `<path>` -- кастомный путь к директории отчёта или `.zip`-архиву (артефакт из CI).
- `--port N` -- альтернативный порт (если 9323 занят).
- `--host 0.0.0.0` -- доступ извне (для просмотра отчёта из другой машины, осторожно: внутренний repo URL).

**Constraints:**

- Команда поднимает локальный http-сервер на ~9323. Печатать URL, не блокировать терминал ожиданием -- предупредить, что для остановки нужен `Ctrl-C` в исходном окне.
- Открытие в браузере на headless-серверах (CI, WSL без GUI) не сработает. На таком окружении использовать `--host` и открыть URL вручную.
- Отчёт ссылается на trace и видео из `test-results/`. Если эта папка удалена -- ссылки в отчёте отдадут 404.
- HTML-отчёт может содержать скриншоты с PII / секретами в URL / cookies -- не публиковать `playwright-report/` в открытом доступе.
