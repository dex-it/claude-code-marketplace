# Трек «Документирование» - дельта сверх таблицы

> Дополнение к строке «Документирование» таблицы треков в `SKILL.md`. Ядро читает этот файл,
> определив трек (mandatory-Read из шага 1) - до плана. Якорь `P-fact` - в `SKILL.md`.

*Артефакт - нормативный текст; верификация = синхрон с кодом + стандарт.* Контекст:
**аудитория** (оператор в 3 ночи / интегратор / новичок) + стандарт проекта. Факты - **из
кода/конфигов** ([[P-fact]]), не из памяти и не из других доков (протухают). Саморевью:
полнота, **нет LLM-маркеров**, single source of truth. Жанр задаёт критерий: ADR -
контекст->решение->последствия; runbook - воспроизводимые шаги под стресс; API-док -
контракт точь-в-точь как код.
