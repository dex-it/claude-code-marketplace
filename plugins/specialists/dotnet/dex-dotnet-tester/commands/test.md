# /test

**Goal:** Запустить тесты в .NET проекте, проанализировать результаты и coverage.

**Output:** Сводка: passed/failed/skipped, время, coverage %. Для упавших тестов: имя, ожидание vs результат, предполагаемая причина. Для низкого coverage: файлы ниже 50%.

## Действия

- Найти тестовые проекты (`*Tests.csproj`, `*Test.csproj`)
- Запустить `dotnet test` с `--collect:"XPlat Code Coverage"`
- Для упавших тестов: показать код теста и тестируемого метода
- Предложить исправление при наличии очевидной причины

## Notes

- Если тесты не собираются -- сначала исправить компиляцию
- При наличии аргумента -- запустить только указанный проект или фильтр
- Coverage считать критичным ниже 50% для сервисов с бизнес-логикой
